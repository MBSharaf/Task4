Struct

Value type → يخزن البيانات مباشرة.

كويس للحاجات الصغيرة والبسيطة.

بيتنسخ لما تعمل نسخة منه.

Class

Reference type → النسخ تشير لنفس المكان في الذاكرة.

كويس للحاجات الكبيرة والمعقدة.

بيدعم inheritance و polymorphism.

Record

Immutable by default → البيانات مش بتتغير بعد الإنشاء.

كويس للمقارنة بين القيم بسهولة.

يسهّل كتابة كائنات تحتوي على بيانات ثابتة.
